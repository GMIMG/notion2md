# notion2md

Automation convert Exported notion markdown(include img, table)

+-----------------------------------+        +--------------+
| Exported notion markdown zip      |        |              |
| (Export format : Markdown & CSV,  +------->+ .md + images |
|  include subpages : On          ) |        |              |
+-----------------------------------+        +--------------+



## Need to

- python3
- Exported zip (Include ONLY ONE .md)
- powershell (or cmd)



## How to

./notion2md.py
./Exprot-*.zip



run


```powershell
./> python3 notion2md
```
